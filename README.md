# BetterHUDVitals
A Road to Vostok mod that replaces vitals texts with icons for more immersion

It also hides conditions that are not active

![illustration](illustration.png)

## Instructions
- Install [VostokMods Mod Loader](https://modworkshop.net/mod/49779)
- Grab the latest release from [here](https://github.com/Rashnain/BetterHUDVitals/releases/latest)
- Move the downloaded ZIP to the `mods` folder in your game installation
